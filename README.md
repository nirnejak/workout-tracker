## Available Scripts

This project supports both `npm` and `bun` as package managers.

**Install Dependencies**

```bash
npm install
# or
bun install
```

**Start Development Server**

```bash
npm run dev
# or
bun dev
```

**Build for Production**

```bash
npm run build
# or
bun run build
```

**Preview Production Build**

```bash
npm run preview
# or
bun run preview
```

**Type Checking**

```bash
npm run type-check
# or
bun run type-check
```

**Linting**

```bash
npm run lint
# or
bun run lint
```

**Fix Linting Issues**

```bash
npm run lint:fix
# or
bun run lint:fix
```

**Format Code**

```bash
npm run format
# or
bun run format
```

**Check Code Formatting**

```bash
npm run format:check
# or
bun run format:check
```
